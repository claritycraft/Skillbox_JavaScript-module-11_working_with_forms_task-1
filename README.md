### En:
#### Task 1
Create a web page with a survey form to collect user data.
During input, some fields are validated for correct user entry. After the form is completed, the user's information is displayed on the screen.

Form fields and functionality:
- Username: A text field where the user must enter their name.
- Email: A field for entering an email address. This text field should validate the correct email format.
- Gender: The user selects their gender from two options — "Male" and "Female" — using radio buttons.
- Service Rating: A slider allowing the user to rate the service from 1 to 10.
- User Interests: The user can select multiple interests from the options: "Sports", "Music", "Travel", "Movies".
- Additional Comments: A text field for entering optional comments.
- Submit Button: When clicked, the user's data should be validated and, if successful, displayed on the page.

- Username and email are required fields.
- Email must match the standard email format.
- Error messages should be shown if the input is invalid.
- Data output: After successful validation, display the user's data on the page, including name, email, gender, service rating, and selected interests.

### Ru:
#### Задача 1
Создайте веб-страницу с формой опроса для сбора данных о пользователе. В момент ввода часть полей проверяется на правильность заполнения пользователем. После заполнения формы информация о пользователе выводится на экран.

Поля и функциональности формы:

Имя пользователя: текстовое поле, где пользователь должен ввести своё имя.
Email: поле для ввода адреса электронной почты. Текстовое поле с валидацией на правильный формат email.
Пол: пользователь выбирает свой пол среди двух вариантов: «Мужской» и «Женский» с использованием радиокнопок.
Оценка сервиса: ползунок, где пользователь может выставить оценку от 1 до 10.
Интересы пользователя: пользователь может выбрать несколько интересов среди опций «Спорт», «Музыка», «Путешествия», «Кино».
Дополнительные комментарии: текстовое поле для ввода дополнительных комментариев.
Кнопка «Отправить»: при нажатии на кнопку данные пользователя должны валидироваться и (при успешной валидации) отображаться на странице.

Валидация формы:

Имя пользователя и email обязательны для заполнения.
Email должен соответствовать стандартному формату email-адреса.
Выводятся сообщения об ошибках, если данные введены неправильно.
Сбор результирующих данных: после успешной валидации формы отобразите данные пользователя на странице, включая имя, email, пол, оценку сервиса и выбранные интересы.
